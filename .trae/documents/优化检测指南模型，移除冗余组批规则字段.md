## 优化检测指南模型

### 问题分析
DetectionGuide 模型中存在冗余的组批规则字段，包括 batch_size, batch_unit, sample_quantity_per_batch, sample_size_min, sample_size_max, sample_size_unit。这些字段已经被正确迁移到了 DetectionItemGuide 关联表中，导致数据冗余和维护困难。

### 解决方案
1. **修改 DetectionGuide 模型**：移除冗余的组批规则字段
2. **更新 to_dict 方法**：移除对冗余字段的引用
3. **更新 API 文档**：移除相关字段的文档说明
4. **更新模型说明文档**：确保文档与实际代码一致

### 具体步骤

#### 1. 修改 DetectionGuide 模型
- 文件：`app/models/detection/detection_guide.py`
- 移除以下字段：
  - batch_size
  - batch_unit
  - sample_quantity_per_batch
  - sample_size_min
  - sample_size_max
  - sample_size_unit

#### 2. 更新 to_dict 方法
- 文件：`app/models/detection/detection_guide.py`
- 移除 to_dict 方法中对上述字段的处理

#### 3. 更新 API 文档
- 文件：`API_DOCUMENTATION.md`
- 移除检测指南相关接口中对组批规则字段的引用，包括：
  - 创建检测指南接口
  - 获取检测指南列表接口
  - 获取单个检测指南接口
  - 更新检测指南接口

#### 4. 更新模型说明文档
- 文件：`app/models/detection/模型说明.md`
- 确保文档准确反映 DetectionGuide 模型的实际结构
- 确认组批规则字段只存在于 DetectionItemGuide 关联表中

### 预期效果
- 消除数据冗余，提高数据一致性
- 明确组批规则的归属，便于维护
- 确保模型设计与业务需求一致
- 提升系统的可扩展性和可维护性
